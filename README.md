# Sitio Web Institucional – Nodo Tecnológico

Este repositorio contiene el desarrollo del sitio web institucional del Nodo Tecnológico, una institución educativa orientada a la formación académica y tecnológica.

El sitio proporciona información sobre la institución, su oferta educativa, los procesos de nuevo ingreso y los medios de contacto para aspirantes y público en general.

**Tecnologías utilizadas:** HTML y CSS.
